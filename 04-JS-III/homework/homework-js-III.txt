Los Arrays:

Un Array, o arreglo, es un tipo de lista que permite almacenar una cantidad de datos de manera ordenada, que luego podran ser accedidos o modificados cuando sea necesario para nuestro programa. Un array puede almacenar todos los tipos de datos como buleanos, cadenas de texto, numeros incluso funciones, y tambien otros arrays dentro de un mismo array. Podemos identificar un array o crearlo usando corchetes [] y dentro de estos corchetes, van todos los elementos del array, separados cada uno por una coma.Un Array, o arreglo, es un tipo de lista que permite almacenar una cantidad de datos de manera ordenada, que luego podran ser accedidos o modificados cuando sea necesario para nuestro programa.
